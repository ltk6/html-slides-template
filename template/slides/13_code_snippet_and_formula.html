Dynamic Weighting Algorithm

Our algorithm dynamically adjusts the relevance score based on contextual parameters. This ensures that edge cases are handled gracefully without manual intervention.

Final Score Calculation
base_score × (1 + context_modifier)
  • base_score: Initial ranking from the semantic search engine.
  • context_modifier: A dynamic float between -0.5 and +0.5 based on real-time user intent signals.
ranker.py 
def calculate_dynamic_score(base_score, intent_signals):
    # Initialize modifier
    modifier = 0.0
    
    # Adjust based on strong semantic matches
    if intent_signals.get("exact_match"):
        modifier += 0.25
        
    # Penalize stale data
    if intent_signals.get("data_age_days") > 30:
        modifier -= 0.15
        
    # Clamp the modifier to allowed range
    final_mod = max(-0.5, min(0.5, modifier))
    
    return base_score * (1.0 + final_mod)
