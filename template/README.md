# Template Reference Gallery

This folder acts as the master library for all 22+ premium slide templates available in this engine.

**Do NOT work directly in this folder.** 
Instead, copy the layouts you need from `template/slides/` over to your active `presentation/slides/` workspace.

### Contents

- **`slides.html`**: A showcase presentation that strings together all 22 slide templates so you can view the entire design system at once.
- **`SLIDE_GUIDE.md`**: A comprehensive design system reference, explaining typography, utility classes, layouts, and HTML structures used throughout the engine.
- **`slides/`**: The raw HTML source code for every slide layout.
